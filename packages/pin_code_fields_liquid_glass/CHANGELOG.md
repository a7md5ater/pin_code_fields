# Changelog

## 0.1.0-dev.1

- Initial development release
- Three visual styles: Separate, Unified, and Blended glass cells
- Full liquid glass renderer settings exposed (blur, thickness, visibility, chromatic aberration, light intensity, etc.)
- Glow effects with configurable radius and colors
- Stretch animations on character entry
- Form validation support via `LiquidGlassPinFormField`
- Built on `pin_code_fields` headless architecture
- Requires Impeller rendering engine (iOS, Android, macOS only)
